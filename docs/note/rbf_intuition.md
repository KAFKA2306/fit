# RBF（放射基底関数）って何？ ~なぜ衣装が「ふわっと」フィットするのか~

ログや説明書に出てくる「RBF」という言葉。
これは数学用語ですが、アーティストの皆さんにとっては「布のような柔らかい変形を作る魔法」だと思ってください。

## 🤖 従来の「硬い」変形

3Dの世界で物を動かすとき、普通は「回転」と「移動」を使います。
これは「箱」を動かすのには適しています。

*   メガネ
*   靴
*   ベルトのバックル

これらは形が変わらないので、そのまま動かせばOKです（これをこのツールでは「SVD」という技術でやっています）。

## 🍮 RBFによる「柔らかい」変形

しかし、服は違います。
キャラクターの胸が大きくなったら、服も「滑らかに」広がる必要があります。
ここで登場するのが RBF (Radial Basis Function) です。

RBFは、「ゴムのシート」をイメージすると分かりやすいです。
いくつかの点を指でつまんで引っ張ったとき、その周りのゴムも自然に伸びますよね？ あの動きを計算で再現しているのです。

### リターゲットでの役割

このシステムでは、以下のように使い分けています：

1.  交差しないパーツ（ボタンなど）: そのまま動かす（硬い動き）
2.  体に密着するパーツ（布地）: RBFで滑らかに変形させる（柔らかい動き）

これにより、「ボタンは歪まないのに、生地だけが体にフィットする」という理想的な結果を自動で作っています。

## 🎨 ユーザーへのメリット

難しい理屈はさておき、RBFのおかげで以下のことが起きます。

*   カクカクしない: 無理やり引き伸ばしたようなポリゴンの歪みが減ります。
*   自然な曲線: 胸や腰のラインに沿って、布が綺麗に流れます。

「なんかこのツール、変形が綺麗だな」と思ったら、裏でRBFが頑張っているんだな、と思い出してください。
