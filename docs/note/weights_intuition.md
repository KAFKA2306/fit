# 透明な「糸」の話：ウェイト転送

リターゲットされた服を着て、アバターが足を上げると、スカートも自然についてきます。
当たり前のように見えますが、実はこれ、すごいことなんです。

## そもそも「ウェイト」って？

3Dモデルの世界では、骨（ボーン）と皮（メッシュ）は別の生き物です。
皮が骨についていくためには、「どの骨に、どれくらい強く引っ張られるか」という指定が必要です。これをウェイト（重み）と呼びます。

例えば、
*   膝の上の皮膚：太ももの骨の影響が100%
*   膝の真ん中：太もも50%、すね50%

といった具合です。

## 完璧なコピー機

このツールが行う「ウェイト転送」は、いわば「透明な糸をつなぐ作業」です。

1.  服の表面にある点をひとつ取ります。
2.  その真下にある「素体の肌」を探します。
3.  「なるほど、君は太ももの骨に70%引っ張られているんだね」と確認します。
4.  その情報を、服の点にそのままコピーします。

これを何万という全ての点に対して、一瞬で行います。

## なぜ綺麗に動くのか

ただコピーするだけではありません。
もし服が肌から浮いていたら？ 足の指先のウェイトが靴に混ざったら？

このシステムは、そういった「ノイズ」を自動で掃除します。
足の指の細かい動きは靴に伝えないようにしたり、誰もいない空間にある頂点は隣から情報を分けてもらったり。

結果として、まるで「見えない磁石」で吸い付いているかのような、自然な追従が実現するのです。
