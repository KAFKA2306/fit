# 「ギザギザ」を消す技術：スムージング

3Dの服を着せてお辞儀をさせたとき、背中やお尻のあたりがカクカクして、鋭利な角が見えてしまったことはありませんか？
これは、服の「折り目」が足りないことが原因です。

## デジタルなアイロンがけ

このツールは、服のデータを見て「ここはよく曲がりそうだな」という場所を自動で探します。
そして、その部分だけ「ポリゴン（3Dの三角形）」を細かく分割します。

普通の紙を丸めると折れてしまいますが、ティッシュのように繊維が細かい紙なら滑らかに曲がりますよね。
それと同じことを、データ上でやっています。

## 必要なところだけやる

全部を細かくすれば綺麗にはなりますが、データが重くなってVRChatなどで敬遠されてしまいます。
なので、このツールは「曲がる関節の近く」で、かつ「ポリゴンが粗い場所」だけをピンポイントで細かくします。

これによって、見た目は滑らかなのに、データ容量は軽いままという、いいとこ取りを実現しています。
