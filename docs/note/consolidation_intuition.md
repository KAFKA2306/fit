# 靴が「くしゃっ」とならない理由：ウェイト統合

アバターの足には、実に多くの骨が入っています。
足首、かかと、足の甲、親指、人差し指...。これらは素足が地面をつかむような動きをするためには必要です。

しかし、革靴やブーツを履いているときに、この動きがそのまま伝わると困ります。
固いはずのブーツが、足の指の動きに合わせてグニャグニャと歪んでしまうからです。

## ミトン理論

そこでこのシステムは、靴のような硬いパーツに対しては「ミトン理論」を使います。

5本指の手袋（素足）は指がバラバラに動きますが、ミトン（靴）は全体がひとかたまりで動きます。
システムは、靴のエリアにある細かい骨（足指など）の影響力をすべて足首などの大きな骨にまとめてしまいます。

## 自動で判断

以前のツールでは、ユーザーが「ここは靴だから固くする」と指定する必要がありました。
しかし今のバージョンでは、先ほど説明した「箱 (OBB)」の仕組みを使って、身体から少し離れている硬そうなパーツを自動で検知し、勝手にこの処理を適用します。

なので、皆さんは何も考えずにリターゲットを実行するだけで、型崩れしない綺麗な靴を手に入れることができます。
