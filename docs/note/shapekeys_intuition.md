# 服が「痩せる」仕組み：シェイプキー同期

VRChatのアバターなどでは、身長を変えたり胸の大きさを変えたりする「シェイプキー（ブレンドシェイプ）」がよく使われます。
素体の胸を大きくしたとき、服が破けてしまったことはありませんか？

このツールを使えば、そんな心配は無用です。

## オートクチュールな自動変形

リターゲット処理が終わった後、ツールは「もしも」のシミュレーションを行います。

*   「もし、素体の胸が100%大きくなったら、この服はどう歪むべきか？」
*   「もし、素体が極限まで痩せたら、この服はどこまで縮むべきか？」

これを計算し、服の方にも同じ名前のシェイプキー（`Breasts_Big` など）を自動生成します。

## 紐づけも全自動で

Unityでアバターの手足を動かすとき、普通は服のシェイプキーも手動で設定する必要があります。
しかし、このツールで作られた服は、最初から素体と同期するように作られています。

あなたが「体型変更ダイヤル」を回せば、服もまるで「魔法の布」のように、そのサイズに合わせて伸縮します。
ボタンひとつで、SサイズからXXLサイズまで自由自在に対応できる服が出来上がるのです。
